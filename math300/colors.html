Colors

We start iwth black and add color, but CMYK is subtractive so we start with white. RGB is more for monitors and CMYK is for printing.

Image formats

GIF - Graphics Image Format. This is a lossless type. Every pixel is specified completely.

JPEG - Joint Photographic Experts Group. This is lossy, compressed format. JPEG1 uses Fourier expansion to compress the image, JPEG2 uses wavelets.

TIFF - Tagged Image File Format. This is sort of a meta-format. it can use other formats, encapsulated with meta-information about those formats. it is favored by actual photographers. Best but not the smallest in size.

PNG - Portable Network Graphics - lossless compressed format .png has replaced .gif pretty much everywhere because it's just better.
